Crestron e-Control 2 has failed to load.

Below is a list of possible solutions.

When a dialog box titled Information Bar appears, a message on the top of the browser window will state "This website wants to install the following add-on: 'Crestron E-Control' from 'Crestron Electronics, Inc'.  If you trust the website and the add-on and want to install it, click here..."  Follow the instructions and click the message and click Install ActiveX Control.  Repeat this process if prompted again.

When a dialog box titled Internet Explorer - Security Warning appears, asking Do you want to install this software? with the name Crestron E-Control  and publisher Crestron Electronics, Inc, click Install.  Repeat this process if prompted again.

In IE, under Tools | Internet Options dialog box, click on the Security tab and make sure your security level has Download signed ActiveX controls set to enable or prompt.
Make sure your user account has either Power User or Administrator privileges.

Make sure your user account has Administrator privileges.

In the Control Panel for Windows, in the Security Center make sure User Account Control is Off.

A version of Microsoft Windows running Microsoft Internet Explorer is required for e-Control 2.
